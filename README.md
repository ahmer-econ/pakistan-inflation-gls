# Determinants of Inflation in Pakistan: A GLS Approach

## Overview
An empirical analysis of the macroeconomic drivers of inflation in Pakistan. This project demonstrates the identification of classical linear regression model (CLRM) violations and the application of Generalized Least Squares (GLS) to ensure reliable estimations.

## Data
- **Source:** State Bank of Pakistan (SBP)
- **Variables:** Consumer Price Index (CPI), Money Supply (M2), Exchange Rate

## Methodology
1. Initial Ordinary Least Squares (OLS) estimation
2. Diagnostic testing for heteroskedasticity and autocorrelation 
3. Application of Generalized Least Squares (GLS) to correct for error term violations
4. Comparative analysis of baseline vs. corrected standard errors

## Software
- **EViews:** Full estimation pipeline, diagnostic testing, and GLS transformation
